# slidev-addon-counter

LaTeX-like multi-level counters and table-of-contents components for Slidev.

## Development

```bash
source ~/.nvm/nvm.sh
nvm use
pnpm install
pnpm check
```

Preview the local addon:

```bash
pnpm dev
```

## Usage

```md
---
addons:
  - slidev-addon-counter
---
```

This repository currently contains the development toolchain and a smoke-test component. Counter components will be added in the next implementation phase.
